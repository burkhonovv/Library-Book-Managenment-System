Project  title: Library Book Record Manager

Project Description 
This project helps the library visitors to search, borrow, and return the books. Library staff can log in with a password to add new books, update the details of existing books, and remove the discarded ones from the system. It also displays contact info of Library staff for further information. The goal of this project is to make an user-friendly,  text based, efficient library management tool.
Main Features / Functionalities
Staff login with password.
Visitor login without password
Add a new book (title, author, year, ID).


Search for a book by ID or title.
Update existing book information.
Delete a book from the records.
Getting basic information about the book borrowers.

C Programming Concepts to Be Used:
File handling for book and user data
Dynamic memory allocation
Pointers
Structures
Functions
Conditional logic for Login system
Expected Output / User Interface
The program will use a text-based menu that provides seperate menu options for Staff and Visitors. Visitors can search, borrow, and return books, while staff can manage book records. All interactions will be shown clearly on the console with structured text menus.
